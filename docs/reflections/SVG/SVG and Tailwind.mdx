# Svg and tailwind

We can use about 200k icons in tailwind with just a config.

And don't stress out, imports are bundled as needed.

```zsh
pnpm i @iconify/json @iconify/tailwind -D
```

and

```Typescript
// tailwind.config.js
const { addDynamicIconSelectors } = require('@iconify/tailwind')
export default {
  plugins: [addDynamicIconSelectors()]
}
```

iconfy icons are always like collection--icon

```tsx
<i class="icon-[material-symbols--arrow-back-ios-new]"></i>
```
